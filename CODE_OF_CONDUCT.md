# Code of Conduct

CGS Sentinel OpenSBIR™ is intended to support constructive, professional, mission-oriented collaboration.

Participants are expected to:

- Communicate respectfully
- Focus on the work and the public mission
- Avoid harassment, discrimination, or abusive conduct
- Respect intellectual property and data-rights boundaries
- Avoid submitting sensitive or proprietary materials to the public repository

Project maintainers may remove comments, issues, or contributions that violate these expectations.
