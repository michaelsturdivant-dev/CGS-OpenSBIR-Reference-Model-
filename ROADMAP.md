# Product Roadmap

## Current Stage: Public Reference MVP

The repository currently provides a public-safe reference model for AI-assisted SBIR, OTA, grant operations, project intake, compliance routing, work-item tracking, artifact planning, and dashboard summaries.

## Phase 1: Demo-Ready Reference Model

Goal: Make the public model easy to understand, run, and explain.

Deliverables:

- Clean README
- Open-core boundary documentation
- Security and data-rights guidance
- Starter FastAPI app
- Public reference endpoints
- Demo script
- Sales offer
- Buyer journey

## Phase 2: Readiness Sprint Toolkit

Goal: Support paid CGS readiness engagements.

Planned deliverables:

- Intake worksheet
- Readiness scorecard
- Problem statement template
- Capability statement template
- Opportunity review template
- Work-item tracker template
- Commercialization pathway template
- Stakeholder map template

## Phase 3: Implementation Toolkit

Goal: Support paid implementation packages.

Planned deliverables:

- Configurable workflow lanes
- Database-backed storage
- Exportable reports
- Admin dashboard concept
- Example deployment guide
- Test suite

## Phase 4: Protected Commercial Extensions

Goal: Keep proprietary CGS Sentinel AI Engine™ capabilities separate from the public repository.

Protected extensions may include:

- Advanced agent routing
- Private prompt chains
- Commercial artifact generation
- Secure customer integrations
- Custom dashboards
- Gmail, calendar, and document workflows
- Client-specific playbooks
- SBIR/OTA strategy logic

## Phase 5: Enterprise and Ecosystem Deployment

Goal: Support larger agencies, universities, contractors, nonprofits, clinics, and community coalitions.

Potential capabilities:

- Multi-tenant hosting
- Organization-level dashboards
- Partner portals
- Compliance review workflow
- Project portfolio management
- Training and technical assistance
- Managed deployment by Closed Gap Solutions
